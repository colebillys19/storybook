# Get Started

DMI Design System is a reusable component library that helps DMI contributors build UIs faster.

The goals is to make building durable UIs more productive and satisfying.

### Install

`npm i @dmi-storybook-design-system`

### Previewing Components

The components are listed alphabetically on the side panel.

To search through the component explorer press the `/` key and enter the component name.

To the left of the page is the side panel with the component groups and their different states.

Located at the top of the page are the `Canvas` and the `Docs` tab.

The Canvas tab is used to preview components while the Docs tabs displays information about the component api and usage.

Below are the `Knobs`, `Story`, `Actions`, and `Accessibility` tabs.

The `Knobs` tab is used to enter changeable props.

The `Story` tab displays the code used to create the component.

The `Actions` displays actions of the component such an onChange and onClick events.

### Use

Import components you want into your UI

`import { BasePaper } from '@dmi-storybook-design-system`

and use them like so

```js
const example = () => {
  const [value, setValue] = React.useState('VALUE');

  const handleChange = (event) => {
    setValue('NEW VALUE');
  };

  return (
    <BasePaper
      onClick={handleClick}
      value={value}
    >
      {value}
    </BasePaper>
  );
};
```
